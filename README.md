# Extreme Disaster Events in Southern Africa (2000–2026)

## Overview
Southern Africa has experienced an increase in extreme weather events such as droughts, floods, and cyclones over the past two decades. This project examines how the frequency and intensity of these events have changed in Southern Africa from 2000–2023. Using UN SDG climate indicators, it explores which countries in the region have been most affected. The goal is to visualize these trends in a way that helps communities and policymakers understand the urgency of climate action
